# Brand

The AgentNest mark combines three protective bands, a runtime boundary, and a bright execution core.
It is designed to remain recognizable at documentation-header and favicon sizes.

## Palette

| Color | Hex | Role |
| --- | --- | --- |
| Agent indigo | `#635BFF` | primary identity |
| Runtime cyan | `#22D3EE` | execution and movement |
| Isolation charcoal | `#101828` | boundary and typography |
| White | `#FFFFFF` | core and negative space |

## Usage

- Preserve clear space equal to one quarter of the mark's width.
- Keep the original proportions and transparent background.
- Use the standalone mark beside a typeset `AgentNest` wordmark; do not place text inside it.
- Do not recolor individual bands, add effects, rotate the mark, or use it as a security certification.

The project asset is [`docs/assets/agentnest-logo.png`](assets/agentnest-logo.png), a 512×512 RGBA PNG.
Contributions that need a vector master should recreate the geometry cleanly rather than tracing
compression artifacts from the PNG.
